A simple and responsive weather application that provides real-time weather data forecasts using the WeatherAPI. It displays current conditions such as temperature, humidity, wind speed, visibility, UV index, and detailed hourly forecasts with a clean and modern UI.

Built with vanilla JavaScript, HTML, and CSS, this project focuses on working with APIs, JavaScript (fetch), and dynamic DOM manipulation.

Features include:

🌡️ Current weather conditions
⏰ Hourly weather updates
🌬️ Wind speed and gust tracking
🌅 Sunrise and sunset times
🌙 Moon phase and illumination
📍 Location-based weather search
🎨 Clean and responsive UI

This project is ideal for learning API integration, handling JSON data, and building real-world frontend applications.
